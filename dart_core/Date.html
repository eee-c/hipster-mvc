Code const Date.fromEpoch(int value, TimeZone timeZone) #

Constructs a new Date instance with the given time zone. The given timeZone must not be null.

This constructor is the only one that doesn't need to be computations and which can therefore be const.

The constructed Date represents 1970-01-01T00:00:00Z + valuems in the given timeZone.

const Date.fromEpoch(int value, TimeZone timeZone);

Code Date changeTimeZone(TimeZone targetTimeZone) #

Returns a new Date in the given targetTimeZone time zone. The value of the new instance is equal to this.value.

This call is equivalent to new Date.fromEpoch(this.value, targetTimeZone).

Date changeTimeZone(TimeZone targetTimeZone);
